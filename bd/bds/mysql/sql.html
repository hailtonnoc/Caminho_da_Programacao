MySQL

O MySQL é um SGBD que se utiliza da linguagem SQL.

Hierarquia da informação

Banco de Dados

Inicialmente temos o banco de dados, onde toda informação ficara contida, este serve para a organização das tabelas.

                        CREATE DATABASE nome ;

                        use nome;

Tabelas

Posteriormente temos as tabelas que servirão para agrupar tipos de registros.
Tabelas e colunas devem seguir alguns parametros como, o nome conter caracteres de a-Z, 0-9. Começando sempre por letras. 

                    CREATE TABLE pessoas
                    (
                        caracteristica_0 varchar (40)
                        caracteristica_1 date
                        caracteristica_3 date
                    );

O que caracterizam as tabelas é o tipo de resistro que a mesma irá manter.

Resistros

As informações são guardadas em registros que contem a maior variação já que cada registro é único, como por exemplos as pessoas.
