Hierarquia da informação

Banco de Dados

E por final temos um pais, com varias culturas que em sua base é constituído por regiões que é formado por pessoas. Por fim temos "banco de dados" base geral ou agrupamento final.

        CREATE DATABASE nome ;

Tabelas

Posteriormente temos a junção de pessoas que formam de uma maneira geral um estado ou uma cultura, esse agrupamento de resistros são realizados em tabelas. Que também pode ser visualizado como pastas.

        CREATE TABLE pessoas
        (
            caracteristica_0 varchar (40)
            caracteristica_1 date
            caracteristica_3 date
        );

Entre os registros se enquadrarem em uma tabela que formam as pastas as mesmas tem que solicitar certos atributos ou características.

Resistros

As informações são guardadas em registros que contem a maior variação já que cada registro é único, como por exemplos as pessoas.
