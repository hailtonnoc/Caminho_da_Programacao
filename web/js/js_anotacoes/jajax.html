Ajax

Faz requisições assíncronas, ou seja, sem travar o JS.

Permite a conexão com servidores

Para solicitar utilize uma new XMLHttpRequest();

                    var ex = new HMLHttpResquest()

Configurando

ex.open("tipo", "valor") abre a conexão da solicitação.

Tipos Função
GET Pegar itens

ex.send() envia a solicitação.

ex.responseText retorna o texto da resposta.
Esse texto de resposta deve ser considerado um JSON que é um tipo de texto para transporte de informações.

JSON.parce('textoResposta') Converte o JSON para Objetos JavaScript.

ex.status Retorna o estado da conexão.

Codigo Significado
200 Bem Sucedido
404 Não encontrado
