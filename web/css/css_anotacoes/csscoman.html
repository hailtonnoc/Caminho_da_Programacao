Comandos

Fontes

Ordem para utilização shothand
font: style weight size, family, outra_family;

Comando Função
font-style: ; Estilo de fonte
font-weight: ; Peso da fonte
font-size: ; Tamanho da fonte
font-family: ; Aplica estilo de fonte

Importando Famílias

É possível enriquecer o site com fontes externas:

Online

<link>

Basta colar os links fornecidos no <head> da pagina HTML e referenciar o nome na font-family do componente desejado.

@import

Referencie o @import no inicio da pagina CSS e declare no font-family do componente desejado.

Arquivo

Para este se fará necessário criar o @font-face

                @font-face
                {
            font-family: 'nome_desejado' ;
            src: url('localização/nome_arquivo.otf') format('formato'), outras url format... ;
            font-weight: ' normal ', ... ;
            font-style: ' normal ', ... ;
                }
Extensão formato
.otf opentype
.ttf truetype
embedded-opentype
truetype-aat
svg

Depois disso basta referenciar com o nome_desejado.
