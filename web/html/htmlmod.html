Formatações

A maneira semântica indica sentido, a não semântica é apenas a forma.

Principais Formatacoes

Negrito / Destaque

Não semântica e pela forma é só usar <b>

Já pela maneira semantica e através da tag Strong

Italico / Enfase

É possível fazer Itálico utilizando a tag <i> (não semantica)

Já pela maneira semantica é simplismente usar a tag <em> que significa emphasis.

Texto marcado

Para marcar um texto usamos a tag <mark>

E para trocar a cor do marcador (não semantico) usamos o estilo

<mark style="background-color:"purple";"> <mark>

Tamanho de textos

Para texto pequeno temos a tag <Small>

Texto deletado

Podemos colocar um texto como Deletado, usando a tag <del>

Texto inserido / enfase ou sublinhado

Basta usar a tag <ins>

É possivel também usar o tag 'U' porem não tem o sentido semântico

Texto Sobrescrito e Subscrito

No texto Sobrescrito temos as letras acima como x'3'+30 = 36 (para cima) tag usada <sup>

No texto Subscrito temos um exemplo H2O (para baixo) tag usada <sub>

Código-fonte

O comando document.getElementById('teste') é escrito em javascript. É conhecido tbm como fonte monoespaçada

Mantendo a Indentação
