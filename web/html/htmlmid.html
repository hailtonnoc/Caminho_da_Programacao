Imagens

Favicon

Imagens que ficam na aba. Deve ser declarado na <head>. Tag link:favicon

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

Imagem na pagina

Para inserir uma imagem basta usar a tag <img>

<img src="link" alt="descricao">

O link pode ser local ou uma URI.

Imagem adaptavel

Ajuda a Ajustar a diferentes tipos de dispositivos

Para isso utilizamos a tag <picture>

Para criar essa analise de tamanho de tela utilize a tag;
<source media="(max-width: tamanhopx )" srcset="local img" type="image/tipo">

Sempre organize os tamanhos em ordem crescente. E a tag <img> é necessaria como imagem inicial.

Audio

Utilize a tag <audio>

Para adicionar menu de reprodução inserir dentro da tag audio a configuração controls.

Opções de carregamento preload

Codigo Função
auto Carrega tudo junto com a pagina
metadata Carrega apenas informações do audio
none Não carrega nada se não clicar

Video

Local em servidor

Não recomentado. Desta maneira consome a banda do servidor.

Para adicionar um video utilize a tag <video>. Assim como devemos adaptar imagens a diferentes tipos de dispositivos, com videos não é diferente.

Devemos utilizar o metodo <source>

Atraves de outros servidores

Youtube

A desvantagem é por se tornar um video publico.
É possivel usar outros sevidores de video, que diminui o custo de hospedagem do cliente.
