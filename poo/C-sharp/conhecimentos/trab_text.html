Variáveis em indices

É possível demarcar variável por índice; 

            class Program
                {
            static void Main(string[] args)
            {
                String txt1, txt2, txt3;
                txt1 = "rato"; txt2 = "rei"; txt3 = "roma";
                Console.WriteLine( "o {0} roeu a roupa do {1} de {2}", txt1,txt2,txt3);
            }
                }

Resultado da execução: >> o rato roeu a roupa do rei de roma
