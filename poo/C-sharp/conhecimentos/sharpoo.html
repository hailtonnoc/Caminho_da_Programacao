Orientação a Objetos

Classes

Classe são tipos de dados compostos por propriedades e métodos. Em resumo classes são objetos compostos onde instanciámos.

Modificador de Classes

Abstract

Sealed

Static

Especificação de Acesso

Public

Sem restrição de acesso.

Private

Só podem ser acessados pela própria classe.

Protected

Podem ser acessados pela própria classe e nas derivadas.

Abstract

Os metodos não podem ser implementados, somente cabeçalhos.

Sealed

Classe não pode ser redefinido.

Virtual

O metodo pode ser redefinido em uma classe derivada.

Static

O metodos pode ser chamado mesmo sem a instanciação de um objeto
