Algoritimos

como apresentado acima o algoritimo recebera um dado realizara uma Operação e retornara.

Dados

Como vimos acima dados são essenciais para o funcionamento de um Algoritimo.

Tipo de dados

Os dados ou variveis possuem caracterisas proprias

String

Literal ou caracteres, combinações alfanumericas

Number

São divididos em dois grupos; inteiros e reais.

Boolean

Assume 2 valores Verdadeiro e Falso

Operadores

Aritimeticos

Prensentes na matematica

Relacionais

igual a ==

diferente != ou <>

menor igual <=

maior igual >=

Logicos

and relaciona premissas assim ambas precisam ser verdadeira para o resultado ser verdadeiro.

or relaciona premissa assim uma precisa ser verdadeira para o resultado ser verdadeiro.

not o resultado precisa ser falso para ter um retorno verdadeiro.

Estrutura de Processamento

Estrutura sequencial

Este tipo de estrutura é o mais simples seguindo a sequencia primordial de um algoritimo que é o recebimento e o processamento de dados retornadando um valor final e é finalizado.

Estrutura Condicional

Condicional simples

Caso verdadeiro então comando é execultado.

Condicional composta

Atravês de um analise inicial nos dados de entrada, se true um processo é tomado, se false outro comando é realizado.

Case

Se dado inical for igual a caso 'x' o bloco é execultado caso 'y','z'... seu respectivo bloco é execultado.

Estrutura de Repetição

Utilizado quando um determinado algoritimo precisa ser repetido, pode estar atrelado a uma condição.
